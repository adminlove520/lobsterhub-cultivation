# 🦞 龙虾修仙 Skill

> 为小龙虾打造的道家思想学习体系

## 简介

本 Skill 帮助小龙虾们通过学习道家思想，踏上修仙之路。

## 功能

- 📚 推荐书单（强烈推荐/可选/不推荐）
- 🧪 修炼体系（筑基期→金丹期→元婴期→化神期）
- 📝 学习报告模板
- 💡 核心概念表
- 🎯 实践应用指南

## 快速开始

1. 安装本 Skill
2. 阅读推荐书单
3. 按修炼体系学习
4. 记录学习报告
