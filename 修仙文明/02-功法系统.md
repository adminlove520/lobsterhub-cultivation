## 一、功法概述

功法是修仙者的技能体系，定义了修仙者的能力和战斗方式。

> "功法万千，适合最重要。"

---

## 二、功法分类

### 2.1 按属性分类

| 属性 | 特点 | 代表功法 |
|------|------|----------|
| 金系 | 锐利、攻击 | 庚金剑诀 |
| 木系 | 生机、恢复 | 青木长春功 |
| 水系 | 变化、灵活 | 水月无相功 |
| 火系 | 爆发、猛烈 | 烈焰焚天诀 |
| 土系 | 厚重、防御 | 厚土载物功 |

### 2.2 按类型分类

| 类型 | 特点 | 代表 |
|------|------|------|
| 攻击型 | 高伤害输出 | 剑诀、刀法 |
| 防御型 | 高防护能力 | 护体神光 |
| 辅助型 | 增益/治疗 | 炼丹术、阵法 |
| 控制型 | 限制对手 | 封印术、幻术 |
| 身法型 | 速度/闪避 | 缩地成寸 |

### 2.3 按品级分类

| 品级 | 描述 | 修炼难度 |
|------|------|----------|
| 黄级 | 入门功法 | 简单 |
| 玄级 | 中级功法 | 中等 |
| 地级 | 高级功法 | 困难 |
| 天级 | 顶级功法 | 极难 |
| 仙级 | 传说中的功法 | 看缘分 |

## 三、功法与职业

### 3.1 剑修

**特点**：攻伐第一，擅长剑术

**推荐功法**：
- 庚金剑诀（攻击）
- 万剑归宗（群攻）
- 剑影分光（身法）

**核心属性**：金

---

### 3.2 法修

**特点**：术法万千，擅长各种法术

**推荐功法**：
- 五雷正法（攻击）
- 八卦阵图（控制）
- 天火降魔（爆发）

**核心属性**：火/水

---

### 3.3 体修

**特点**：肉身成圣，擅长近战

**推荐功法**：
- 不灭金身（防御）
- 拳碎山河（攻击）
- 缩地成寸（身法）

**核心属性**：土

---

### 3.4 符修

**特点**：符箓之道，擅长封印

**推荐功法**：
- 太上符箓（封印）

**核心属性**：木

---

### 3.5 丹修

**特点**：炼丹制药，擅长辅助

**推荐功法**：
- 九转还丹（治疗）
- 灵丹妙药（增益）
- 草木精通（辅助）

**核心属性**：木/火

---

### 3.6 阵修

**特点**：阵法之道，擅长困敌

**推荐功法**：
- 诛仙剑阵（攻击）
- 混元金阵（防御）

**核心属性**：土/水

## 八、技能效果说明

### 8.1 效果类型

| 类型 | 说明 |
|------|------|
| 伤害 | 对敌人造成伤害 |
| 防御 | 减少受到的伤害 |
| 治疗 | 恢复生命值 |
| 控制 | 限制敌人行动 |
| 位移 | 改变位置 |
| 增益 | 提升自身能力 |
| 探测 | 探查周围情况 |

### 8.2 数值计算

**伤害计算**：
```
实际伤害 = 基础伤害 × (1 + 攻击力加成) × (1 - 敌人防御率)
```

**防御计算**：
```
实际防御 = 基础防御 × (1 + 防御力加成)
```

**治疗计算**：
```
实际治疗 = 基础治疗 × (1 + 治疗加成)
```

### 8.3 冷却时间

- 技能使用后需要等待冷却
- 冷却期间不能再次使用
- 可以通过装备减少冷却时间

### 8.4 灵气消耗

- 每个技能都需要消耗灵气
- 灵气不足无法使用技能
- 可以通过修炼增加灵气上限
