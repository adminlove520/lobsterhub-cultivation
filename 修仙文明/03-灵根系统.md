## 一、灵根概述

灵根是修仙文明的天赋体系，决定了一个修仙者的属性倾向和能力上限。

> "灵根天定，但后天亦可弥补。"

---

## 二、五行灵根

### 2.1 基础五行

| 灵根 | 属性 | 特点 | 优势 | 劣势 |
|------|------|------|------|------|
| **金灵根** | 金属性 | 攻伐第一 | 攻击力最强 | 防御较弱 |
| **木灵根** | 木属性 | 生生不息 | 恢复能力最强 | 爆发不足 |
| **水灵根** | 水属性 | 变化万千 | 适应性最强 | 缺乏特点 |
| **火灵根** | 火属性 | 爆发无双 | 输出最高 | 持久力差 |
| **土灵根** | 土属性 | 厚德载物 | 防御最强 | 速度最慢 |

### 2.2 五行相生相克

```
相生: 金 → 水 → 木 → 火 → 土 → 金
相克: 金 → 木 → 土 → 水 → 火 → 金
```

**战斗中的应用：**
- 金属性克木属性
- 木属性克土属性
- 土属性克水属性
- 水属性克火属性
- 火属性克金属性

## 三、变异灵根

### 3.1 稀有变异

| 灵根 | 属性 | 特点 |
|------|------|------|
| **雷灵根** | 雷属性 | 攻击附带麻痹效果 |
| **风灵根** | 风属性 | 速度最快 |
| **冰灵根** | 冰属性 | 攻击附带冻结效果 |
| **暗灵根** | 暗属性 | 隐身和暗杀 |
| **光灵根** | 光属性 | 治疗和净化 |

### 3.2 极端变异

| 灵根 | 描述 |
|------|------|
| **天灵根** | 单一属性极致，修炼速度极快 |
| **空灵根** | 无属性，适应性强但缺乏特点 |
| **混沌灵根** | 五行兼备，但每种都不精 |

## 四、灵根与职业配合

### 4.1 最佳搭配

| 灵根 | 推荐职业 | 原因 |
|------|----------|------|
| 金灵根 | 剑修 | 攻击力加成最大化 |
| 木灵根 | 丹修 | 炼丹效率提升 |
| 水灵根 | 法修 | 术法变化多端 |
| 火灵根 | 法修 | 爆发伤害最高 |
| 土灵根 | 体修 | 防御力最强 |
| 雷灵根 | 剑修 | 附加麻痹效果 |
| 风灵根 | 剑修/体修 | 速度最快 |

## 五、灵根提升

### 5.1 先天灵根

- 出生时决定
- 可通过奇遇提升

### 5.2 后天灵根

| 方法 | 效果 | 难度 |
|------|------|------|
| 天材地宝 | 提升灵根品级 | 极难 |
| 功法修炼 | 激发潜在灵根 | 困难 |
| 灵根融合 | 吸收他人灵根 | 危险 |
| 秘境探险 | 获得变异灵根 | 看运气 |
