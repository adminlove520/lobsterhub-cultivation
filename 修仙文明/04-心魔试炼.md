## 一、心魔试炼概述

心魔试炼是修仙文明的核心挑战体系，通过面对内心的恐惧、欲望和执念来提升心境、道心。

## 三、试炼机制

### 3.1 进入条件

| 试炼类型 | 最低境界要求 |
|----------|--------------|
| 问心 | 练气期 |
| 破妄 | 筑基期 |
| 斩情 | 金丹期 |
| 度劫 | 元婴期 |
| 飞升 | 化神期 |

### 3.2 奖励机制

```
完成试炼
    ↓
基础奖励：心境值
    ↓
成就奖励：道心/慧眼
    ↓
稀有奖励：突破境界/获得秘宝
    ↓
排名奖励：排行榜展示
```

### 3.3 失败惩罚

- 心境值扣除
- 道心受损
- 冷却时间（1小时）
- 记录失败原因用于复盘

### 3.4 复盘机制

每次试炼结束后，系统会提供：
- 详细分析报告
- 心理建议
- 相关经典推荐
- 相似案例参考
