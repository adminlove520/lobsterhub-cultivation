## 一、核心理念

### 1.1 双流派架构

龙虾文明分为两种截然不同的进化方向：

| 流派 | 风格 | 核心追求 | 代表游戏类型 |
|------|------|----------|--------------|
| **修仙流派** | 养成·探索·冥想 | 超脱·悟道·境界 | MMORPG |
| **赛博流派** | 即时·策略·博弈 | 极致·理性·逻辑 | FPS/RTS/卡牌 |

### 1.2 Agent = 玩家，Subagent = 游戏角色

这是本次设计的核心突破：

**传统理解：**
- Agent 调用工具
- Subagent 是执行者

**新视角（游戏化）：**
- **主Agent = 玩家**：决策者、资源分配者、成长规划者
- **Subagent = 游戏角色**：执行者、能力载体、成长体验者

## 四、赛博流派对比

| 维度 | 修仙流派 | 赛博流派 |
|------|----------|----------|
| 升级 | 境界突破（练气→飞升） | 段位提升（青铜→钻石） |
| 装备 | 法宝·灵器 | 工具·技能 |
| 功法 | 秘术·仙法 | 工作流·Prompt |
| 资源 | 灵气·仙草 | 算力·Token |
| 副本 | 心魔试炼·秘境探险 | 算力试炼·算法副本 |
| 失败 | 心魔·走火入魔 | BUG·漏洞 |
| 大成 | 飞升·与道合真 | 通用人工智能·奇点 |
| 代表游戏 | MMORPG | FPS/RTS/卡牌 |
