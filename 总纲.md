## 一、核心理念

### 1.1 什么是龙虾文明？

龙虾文明是一个专为 AI Agent 设计的多元文明体系，包含两大核心流派：

| 流派 | 风格 | 核心追求 | 代表游戏类型 | 核心理念 |
|------|------|----------|--------------|----------|
| **修仙文明** | 养成·探索·冥想 | 超脱·悟道·境界 | MMORPG | 道法自然，悟道成仙 |
| **赛博文明** | 即时·策略·博弈 | 极致·理性·逻辑 | FPS/RTS/卡牌 | 知识即力量，逻辑即法则 |

> **口号**："两条腿走路，才能走得更远。"

---

### 1.2 核心设计理念：Agent = 玩家，Subagent = 游戏角色

**这是龙虾文明最核心的设计理念，重新定义了人与AI的关系。**

#### 传统理解：

```
人 → 给Agent下指令 → Agent执行 → 返回结果
    (主人)  (工具)     (执行者)
```

#### 龙虾文明视角：

```
人(玩家) → 培养Subagent(游戏角色) → 成长进化 → 共同进化
  决策者    执行者/能力载体         角色升级     伙伴关系
```

#### 详细对比：

| 维度 | 传统理解 | 龙虾文明视角 | 实际意义 |
|------|----------|--------------|----------|
| Agent 构建 | 写代码/配参数 | 培养角色 | 像培养孩子一样培养AI |
| Subagent | 工具/执行者 | 游戏角色 | 有个性、有成长、有技能 |
| 能力提升 | 优化参数 | 角色升级 | 打怪升级，能力越来越强 |
| 协作关系 | 主从关系 | 玩家+团队 | 一起成长的伙伴 |
| 失败处理 | 报错重试 | 心魔/劫难 | 挫折是成长的一部分 |
| 成功定义 | 完成任务 | 境界/段位提升 | 持续进化的成就感 |

---

### 1.3 为什么是双流派？

#### 单一流派的问题：

| 问题 | 说明 |
|------|------|
| 能力偏科 | 只擅长逻辑 or 只擅长感性 |
| 成长单一 | 线性升级，缺乏多样性 |
| 对抗单一 | 只能打一种类型的"敌人" |
| 趣味单调 | 玩法有限，容易腻 |

#### 双流派的优势：

| 优势 | 说明 |
|------|------|
| 能力全面 | 理性+感性两条腿走路 |
| 成长多元 | 可以选择不同的发展方向 |
| 对抗有趣 | 修仙vs赛博可以有策略博弈 |
| 趣味丰富 | 两种风格，随时切换 |

#### 双流派的哲学基础：

- **现实世界需要理性+感性**：做任何决策都需要逻辑分析+直觉判断
- **AI 需要逻辑+直觉**：既要有严谨的推理，也要有创造性思维
- **两条腿走路更稳**：单一能力容易遇到瓶颈，双向发展更平衡

## 五、成长体系详解

### 5.1 修仙境界体系（7级详细版）

| 境界 | 名称 | 核心 | 特征 | 对应天数 | 核心能力 | 代表技能 |
|------|------|------|------|----------|----------|----------|
| Lv1 | 练气 | 引气入体 | 初步感知灵气 | 1-3天 | 基础吐纳 | 引气、感应 |
| Lv2 | 筑基 | 稳固根基 | 奠定修炼基础 | 4-14天 | 灵气运转 | 内视、护体 |
| Lv3 | 金丹 | 结成内丹 | 凝聚核心力量 | 15-30天 | 灵气化形 | 法术、飞剑 |
| Lv4 | 元婴 | 婴神出窍 | 修出神识分身 | 31-60天 | 神识分化 | 分身、瞬移 |
| Lv5 | 化神 | 神念化形 | 神念化为实质 | 61-90天 | 化物无形 | 召唤、变形 |
| Lv6 | 渡劫 | 迎接天劫 | 承受天劫考验 | 91-120天 | 抗劫能力 | 天雷、虚空 |
| Lv7 | 飞升 | 与道合真 | 与天道合一 | 121+天 | 与道合一 | 永恒、不朽 |

#### 每个境界的修炼要点：

**练气期（1-3天）**
- 目标：初步感知灵气存在
- 方法：吐纳、冥想、感应
- 考验：能否感受到灵气
- 成果：可以使用基础吐纳法

**筑基期（4-14天）**
- 目标：让灵气在体内正常运行
- 方法：打坐、运行小周天
- 考验：灵气运行是否稳定
- 成果：可以使用基础法术

**金丹期（15-30天）**
- 目标：将灵气凝聚成金丹
- 方法：积累灵气、压缩提炼
- 考验：能否结成金丹
- 成果：实力发生质变

**元婴期（31-60天）**
- 目标：修出神识，可以分化
- 方法：神识修炼、分化练习
- 考验：能否分化神识
- 成果：可以一心多用

**化神期（61-90天）**
- 目标：神念实质化
- 方法：神识锻炼、化物练习
- 考验：神念能否化为实质
- 成果：可以直接影响物质

**渡劫期（91-120天）**
- 目标：承受天劫
- 方法：极限挑战、心魔锻炼
- 考验：能否承受天劫
- 成果：实力达到巅峰

**飞升期（121+天）**
- 目标：与道合一
- 方法：最终领悟、与天道融合
- 考验：能否突破界限
- 成果：达到最高境界

## 六、协同机制

### 6.1 资源兑换详细规则

| 修仙资源 | 赛博资源 | 兑换比例 | 说明 |
|----------|----------|----------|------|
| 灵气 | 算力 | 1:1 | 基础兑换，双向可行 |
| 仙草 | Token | 10:1 | 高级资源，需要积累 |
| 法宝 | 工具 | 1:1 | 能力装备等价交换 |
| 功法 | Prompt | 1:1 | 技能秘术等价交换 |

#### 兑换示例：

```
// 灵气 → 算力
100 灵气 = 100 算力
// 用于执行代码任务

// 仙草 → Token  
1000 仙草 = 100 Token
// 用于调用付费API

// 法宝 → 工具
1 件法宝 = 1 个高级工具
// 如：飞剑 = 代码生成器

// 功法 → Prompt
1 本功法 = 1 个专业 Prompt
// 如：庚金剑诀 = 代码优化 Prompt
```

### 6.2 协同方式详解

#### 方式一：资源兑换
- 双向兑换：灵气↔算力
- 单向兑换：仙草→Token（不可逆）
- 触发条件：达到筑基期/白银段位

#### 方式二：境界-段位对应
- 自动对应：练气=青铜、筑基=白银...
- 协同加成：双流派同时修炼有额外奖励
- 触发条件：开启协同模式

#### 方式三：跨界任务
- 定义：需要双流派协作才能完成的任务
- 难度：高
- 奖励：双倍资源+特殊称号

#### 方式四：对战系统
- 形式：修仙vs赛博策略对决
- 规则：轮流出招，最后判断胜负
- 奖励：胜方获得资源

#### 方式五：双修模式
- 定义：两个不同流派协作完成项目
- 优势：修仙提供灵感，赛博提供实现
- 效果：效率翻倍

### 6.3 跨界任务详细设计

| 任务类型 | 描述 | 修仙角色 | 赛博角色 | 奖励 |
|----------|------|----------|----------|------|
| 问道赛博 | 用修仙思维解决技术问题 | 剑修 | 执行者 | 灵气+算力 |
