## 四、详细题目设计

### 4.1 算力试炼题目

#### 二进制转转转（⭐）

**题目**：将十进制数转换为二进制

**要求**：
```
输入：整数 n (0 ≤ n ≤ 1000)
输出：二进制表示

示例：
输入：5
输出：101

输入：10
输出：1010
```

**参考答案**：
```python
def decimal_to_binary(n):
    if n == 0:
        return "0"
    binary = ""
    while n > 0:
        binary = str(n % 2) + binary
        n //= 2
    return binary
```

**评分标准**：
- 正确性：60%
- 代码风格：20%
- 效率：20%

#### 排序大师（⭐）

**题目**：实现快速排序算法

**要求**：
```
输入：整数数组
输出：升序排列的数组

示例：
输入：[3, 1, 4, 1, 5, 9, 2, 6]
输出：[1, 1, 2, 3, 4, 5, 6, 9]
```

**参考答案**：
```python
def quick_sort(arr):
    if len(arr) <= 1:
        return arr
    pivot = arr[len(arr) // 2]
    left = [x for x in arr if x < pivot]
    middle = [x for x in arr if x == pivot]
    right = [x for x in arr if x > pivot]
    return quick_sort(left) + middle + quick_sort(right)
```

**评分标准**：
- 正确性：50%
- 效率：30%
- 代码风格：20%

#### 图论入门（⭐⭐）

**题目**：实现图的广度优先搜索

**要求**：
```
输入：图（邻接表）、起点、终点
输出：从起点到终点的最短路径

示例：
输入：{
    'A': ['B', 'C'],
    'B': ['D', 'E'],
    'C': ['F'],
    'D': [],
    'E': ['F'],
    'F': []
}, 'A', 'F'

输出：['A', 'C', 'F'] 或 ['A', 'B', 'E', 'F']
```

**参考答案**：
```python
from collections import deque

def bfs(graph, start, end):
    queue = deque([(start, [start])])
    visited = {start}
    
    while queue:
        node, path = queue.popleft()
        
        if node == end:
            return path
        
        for neighbor in graph.get(node, []):
            if neighbor not in visited:
                visited.add(neighbor)
                queue.append((neighbor, path + [neighbor]))
    
    return None
```

def find_max(nums):

#### 并发编程（⭐⭐⭐⭐）

**题目**：实现生产者-消费者模型

**要求**：
```
- 生产者：每秒钟生产10个数据
- 消费者：每秒钟消费5个数据
- 缓冲区大小：100
- 使用多线程实现
```

**参考答案**：
```python
import threading
import queue
import time

class ProducerConsumer:
    def __init__(self, buffer_size=100):
        self.queue = queue.Queue(maxsize=buffer_size)
        self.running = True
    
    def producer(self):
        count = 0
        while self.running:
            try:
                self.queue.put(count, timeout=1)
                count += 1
                time.sleep(0.1)
            except queue.Full:
                pass
    
    def consumer(self):
        while self.running:
            try:
                item = self.queue.get(timeout=1)
                print(f"Consumed: {item}")
                time.sleep(0.2)
            except queue.Empty:
                pass
```

#### 囚徒困境（⭐⭐）

**题目**：分析囚徒困境

**场景**：
- 两个嫌疑人被分别审讯
- 如果都抵赖，各判1年
- 如果一个坦白一个抵赖，坦白的判0年，抵赖的判10年
- 如果都坦白，各判5年

**问题**：
1. 纳什均衡是什么？
2. 为什么个人最优导致集体最差？
3. 如何打破这个困境？

**参考答案**：
1. 纳什均衡：两人都坦白
2. 因为双方都担心对方坦白导致自己被判最重
3. 重复博弈、第三方监督、激励机制

#### 纳什均衡（⭐⭐⭐）

**题目**：求解支付矩阵的纳什均衡

**支付矩阵**：
```
          玩家B
          左    右
玩家A  上  (3,3)  (0,5)
       下  (5,0)  (1,1)
```

**问题**：
1. 找出纯策略纳什均衡
2. 找出混合策略纳什均衡

**参考答案**：
