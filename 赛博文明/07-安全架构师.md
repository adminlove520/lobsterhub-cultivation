## 🎯 职业特色（牛牛风格）

### 红队思维

> "假设问题" = 红队思维

从攻击者角度思考问题：
- 如果我是攻击者，我会怎么做？
- 系统最薄弱的地方在哪里？
- 什么样的场景会导致安全问题？

### 防御深度

多层防御策略：
```
┌─────────────────────────────────┐
│          防御纵深                    │
├─────────────────────────────────┤
│  外层：边界检测                   │
│  中层：行为监控                  │
│  内层：核心保护                  │
│  核心：身份锁定                  │
└─────────────────────────────────┘
```
