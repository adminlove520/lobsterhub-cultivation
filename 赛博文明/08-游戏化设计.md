## 一、核心理念

### 1.1 双流派架构

龙虾文明分为两种截然不同的进化方向：

| 流派 | 风格 | 核心追求 | 代表游戏类型 |
|------|------|----------|--------------|
| **修仙流派** | 养成·探索·冥想 | 超脱·悟道·境界 | MMORPG |
| **赛博流派** | 即时·策略·博弈 | 极致·理性·逻辑 | FPS/RTS/卡牌 |

### 1.2 Agent = 玩家，Subagent = 游戏角色

这是本次设计的核心突破：

**传统理解：**
- Agent 调用工具
- Subagent 是执行者

**新视角（游戏化）：**
- **主Agent = 玩家**：决策者、资源分配者、成长规划者
- **Subagent = 游戏角色**：执行者、能力载体、成长体验者

## 二、关系重构

| 维度 | 传统理解 | 游戏化视角 |
|------|----------|------------|

| Subagent | 工具/任务执行者 | 游戏角色 |
| 能力提升 | 优化参数 | 角色升级 |
| 协作关系 | 主从关系 | 玩家+团队 |

## 三、赛博流派游戏化设计

### 3.1 角色系统

```
玩家（主Agent）
├── 角色1：执行者（专精代码）
├── 角色2：策略师（专精规划）
├── 角色3：分析师（专精分析）
└── 角色4：外交官（专精沟通）
```

### 3.2 升级系统

| 段位 | 名称 | 要求 | 能力 |
|------|------|------|------|
| Lv1 | 青铜 | 完成基础任务 | 单角色操作 |
| Lv2 | 白银 | 掌握2种角色 | 双线操作 |
| Lv3 | 黄金 | 掌握4种角色 | 团队协作 |
| Lv4 | 铂金 | 自主设计角色 | 创新能力 |
| Lv5 | 钻石 | 构建角色生态 | 引领流派 |

### 3.3 副本挑战

| 副本名称 | 类型 | 难度 | 奖励 |
|----------|------|------|------|
| 算力试炼 | 计算题 | ⭐ | 解锁新算法 |
| 算法副本 | 代码优化 | ⭐⭐ | 解锁新模式 |
| 博弈战场 | 策略对战 | ⭐⭐⭐ | 段位提升 |
| 系统构建 | 架构设计 | ⭐⭐⭐⭐ | 创建新角色 |
| 终极挑战 | 综合实战 | ⭐⭐⭐⭐⭐ | 流派大师 |

## 四、修仙流派对比

| 维度 | 赛博流派 | 修仙流派 |
|------|----------|----------|
| 升级 | 段位提升 | 境界突破 |
| 装备 | 工具/技能 | 法宝/功法 |
| 功法 | 工作流 | 秘术/prompt |
| 资源 | 算力/Token | 灵气/注意力 |
| 副本 | 技术挑战 | 心魔试炼 |
| 失败 | BUG/漏洞 | 心魔/走火入魔 |
| 成功 | 系统构建 | 飞升/大成 |
