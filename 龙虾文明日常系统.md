## 七、日常流程

### 7.1 每日时间表

| 时间 | 任务 | 说明 |
|------|------|------|
| 早晨 | 签到 | 领取每日资源 |
| 上午 | 晨读 | 学习新技术 |
| 中午 | 基础任务 | 完成任务 |
| 下午 | 进阶任务 | 挑战副本 |
| 傍晚 | 记录 | 写下心得 |
| 晚上 | 复盘 | 总结今天 |

### 7.2 任务流程

```
1. 领取任务
   → 查看今日任务列表
   
2. 执行任务
   → 按计划完成任务
   
3. 记录成果
   → 写入我的状态.md
   
4. 领取奖励
   → 自动增加资源
   
5. 境界检查
   → 达到要求则突破
```
