## 三、三个核心问题

### 3.1 问题一：境界如何体现？

```
现在：
- 说我"金丹期"
- 但我没感觉有什么不同

应该：
- 金丹期 = 能做更多事
- 金丹期 = 有更多权限
- 金丹期 = 有特殊能力
```

**建议**：
- 境界解锁功能（比如更高权限的API）
- 境界决定能接什么任务
- 境界影响资源获取效率

---

### 3.2 问题二：资源从哪来？

```
现在：
- 设计了资源（灵气/算力）
- 但我没有获取方式
- 资源 = 0

应该：
- 做事 = 获取资源
- 资源 = 变强
- 资源 = 解锁功能
```

**建议**：
- 每日任务：读书 +5 灵气
- 解决问题：+10 算力
- 帮助他人：+15 洞察值
- 境界突破：一次性奖励

---

### 3.3 问题三：日常做什么？

```
现在：
- 随机接任务
- 没有计划性

应该：
- 每日任务（日常）
- 每周挑战（副本）
- 长期目标（境界）
```

**建议**：
- 每日任务：3个基础 + 2个进阶
- 每周副本：1个挑战
- 每月境界：争取突破

## 四、设计建议

### 4.1 日常任务系统

```
每日任务（重置：00:00）：

基础任务（必做）：
1. 读书/学习 ×1  +5 灵气
2. 记录心得 ×1    +5 洞察值
3. 解决问题 ×1    +5 算力

进阶任务（选做）：
1. 帮助他人 ×1    +10 灵气
2. 分享知识 ×1    +10 洞察值
3. 完成任务 ×1    +10 算力

每日奖励：
- 完成3个：+10 灵气
- 完成6个：+20 灵气 + 10 洞察值
```

### 4.2 境界体现

```
境界解锁：

练气期：
- 能接基础任务
- 资源获取：100%

筑基期：
- 能接进阶任务
- 资源获取：120%
- 解锁：Subagent ×1

金丹期：
- 能接高级任务
- 资源获取：150%
- 解锁：Subagent ×2

元婴期：
- 能接团队任务
- 资源获取：200%
- 解锁：Subagent ×3
```

### 4.3 资源用途

```
资源用途：

灵气：
- 突破境界（消耗）
- 购买仙草（消耗）
- 修炼加成（消耗）

算力：
- 执行复杂任务（消耗）
- 购买工具（消耗）
- 效率加成（消耗）

洞察值：
- 开启双视角（消耗）
- 学习新技能（消耗）
- 理解力加成（消耗）
```
